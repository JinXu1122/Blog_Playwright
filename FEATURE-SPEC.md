# 博客系统功能规格说明书

## 1. 项目概述

- **项目名称**: Next.js 全栈博客系统
- **技术栈**: Next.js 16 (App Router) + SQLite + Drizzle ORM + Tailwind CSS
- **项目路径**: `blog-proj/`
- **数据库**: SQLite (`blog.db`)

---

## 2. 数据模型

### 2.1 文章 (posts)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| title | TEXT | 必填，文章标题 |
| summary | TEXT | 必填，文章摘要 |
| content | TEXT | 必填，文章正文 |
| categoryId | INTEGER | 关联分类，可为空 |
| createdAt | TIMESTAMP | 创建时间，自动生成 |
| viewCount | INTEGER | 阅读计数，默认 0 |

### 2.2 分类 (categories)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| name | TEXT | 分类名称，唯一 |
| slug | TEXT | URL 友好的标识，唯一 |

**默认分类**: 技术 (tech)、生活 (life)、随想 (thoughts)

### 2.3 评论 (comments)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| postId | INTEGER | 关联文章 |
| authorName | TEXT | 评论者姓名 |
| content | TEXT | 评论内容 |
| createdAt | TIMESTAMP | 创建时间 |

---

## 3. 功能列表

### 3.1 文章管理
- [x] **文章列表页** (`/`)
  - 展示所有文章
  - 显示标题、摘要、发布日期、分类、阅读数
  - 支持删除文章
  - 支持按分类筛选
  - 支持关键词搜索

- [x] **文章详情页** (`/posts/[id]`)
  - 显示完整文章内容
  - 显示发布日期、阅读数、分类
  - 显示评论列表
  - 支持发表评论
  - 阅读时自动增加阅读计数

- [x] **创建文章页** (`/create`)
  - 表单填写：标题、摘要、内容、分类
  - 表单验证
  - 创建成功后跳转首页

### 3.2 分类系统
- [x] **分类系统**
  - 预设三个分类：技术、生活、随想
  - 文章可选择一个分类
  - 支持按分类筛选文章列表

### 3.3 评论系统
- [x] **发表评论**
  - 填写姓名和评论内容
  - 提交后实时更新评论列表
  - 不触发文章阅读数增加

- [x] **评论列表**
  - 显示所有评论
  - 显示评论者姓名、内容、时间

### 3.4 搜索功能
- [x] **关键词搜索**
  - 搜索框位于首页顶部导航栏
  - 按文章标题搜索
  - 搜索结果页显示「搜索结果: "xxx"」
  - 点击其他筛选条件时自动清除搜索框文本

## 4. API 接口

### 4.1 文章接口

| 方法 | 路径 | 说明 | 参数 |
|------|------|------|------|
| GET | `/api/posts` | 获取文章列表 | `?q=关键词`, `?category=slug` |
| POST | `/api/posts` | 创建文章 | `title`, `summary`, `content`, `categoryId` |
| GET | `/api/posts/[id]` | 获取文章详情 | - |
| DELETE | `/api/posts/[id]` | 删除文章 | - |

### 4.2 其他接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/categories` | 获取所有分类 |
| POST | `/api/comments` | 创建评论 |

---

## 5. 页面路由

| 路径 | 说明 | 渲染方式 |
|------|------|----------|
| `/` | 首页，文章列表 | 静态 + 客户端筛选 |
| `/posts/[id]` | 文章详情页 | 动态渲染 |
| `/create` | 创建文章页 | 静态渲染 |

## 7. 用户交互流程

### 7.1 阅读文章
1. 用户在首页点击文章标题
2. 跳转至文章详情页
3. 阅读数自动 +1
4. 用户可查看分类、评论

### 7.2 发表评论
1. 用户在文章详情页填写姓名和评论
2. 点击「发表评论」
3. 评论立即显示在列表中
4. **不触发阅读数增加**

### 7.3 搜索文章
1. 用户在搜索框输入关键词
2. 点击搜索按钮或按回车
3. 跳转至搜索结果页
4. 显示匹配标题的文章

### 7.4 筛选文章
1. 用户点击分类链接（技术/生活/随想）
2. 显示该分类下的所有文章
3. 搜索框文本自动清除
4. 点击「清除筛选」返回全部文章

### 7.5 写文章
1. 用户点击顶部「写文章」按钮
2. 跳转至创建文章页 `/create`
3. 填写标题、摘要、内容
4. （可选）选择一个分类
5. 点击「发布文章」
6. 跳转至首页，新文章显示在列表顶部
